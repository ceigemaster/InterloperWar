These .cs (C++ Script) files will help you learn the basics of using the Torque2D engine.
The guides are in .cs format in order to familiarize you with the format.
Most of the instructions will be commented out.
It is not recommended to actually execute the scripts.
They are useless for actual execution within the engine.
To test any of the concepts you learn here, start the engine executable (DarkInterlopers.exe) and press Ctrl+~
